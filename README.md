## These are the features of this app:

-   [x] Draw Lines
-   [x] Draw by Pen
-   [x] Insert image to the draw
-   [x] Dark / Light theme switch
-   [x] Change the pointer size
-   [x] Change the color of the pointer
-   [x] Reset and Resize the workspace
