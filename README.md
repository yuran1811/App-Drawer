## Features

- [x] Infinite workspace (can be modified)
- [x] Draw freestyle
- [x] Draw lines
- [x] Draw circles
- [x] Draw rectangles (fill | nofill)
- [x] Draw images
- [x] Theme toggle
- [x] Change brush size
- [x] Change brush color
- [x] Reset & Resize the workspace
- [x] Undo & Redo draw
